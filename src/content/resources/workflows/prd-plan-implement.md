# 文档链

1. **PRD**：问题、用户、成功标准、非目标
2. **技术方案**：模块边界、数据、风险、备选
3. **Tickets**：可独立合并的任务切片
4. **Plan**：本轮具体步骤与验证
5. **Implement**：按切片交付，每片可验收

原则：没有验收标准的 Ticket 不进入 Implement。
