# 智面官 OfferPilot

> 你的专属 AI 面试陪练官

## 一句话价值
智面官（OfferPilot）是一款基于仓颉语言开发的 AI 面试陪练智能体，面向求职训练场景，支持岗位定制、难度选择、面试官风格切换、动态追问、表现评估、总结报告、弱项识别与训练计划生成。

## 当前版本
V3 强化追问与会话版

## 当前功能
- 主菜单
- 完整模拟面试
- 专项训练模式
- 岗位选择
- 难度选择
- 面试官风格切换
- 逐题作答
- 一轮规则版动态追问
- 多维评分
- 文本条形图报告
- 弱项识别
- 历史成绩持久化与趋势对比
- 本次分数 vs 历史平均分
- 7天训练计划
- 中途退出
- 会话摘要
- 报告后弱项直达专项训练
- 题库配置化加载（`docs/question_bank.cfg`）

## 当前支持岗位
- 后端开发
- 前端开发
- 产品经理

## 运行方式
1. 使用仓颉环境运行：
```bash
cjpm run
```
2. 推荐使用自动清理脚本（避免 `main.exe` 占用）：
```powershell
.\run.ps1
```
